Materi HTML Basic

  1. Mengenal lebih dekat dengan HTML
    1. Pengertian HTML

      2. HTML (HyperText Markup Language) adalah bahasa markah yang digunakan untuk membuat dan menyusun struktur konten pada halaman web. HTML berfungsi sebagai kerangka dasar dari suatu halaman web dan digunakan untuk menentukan elemen-elemen seperti teks, gambar, tautan, video, dan lainnya, serta bagaimana elemen-elemen tersebut saling terkait. Secara singkat, HTML adalah bahasa markah yang digunakan untuk mengatur struktur konten halaman web. Seiring waktu, HTML mengalami perkembangan dari versi awal hingga mencapai HTML5, yang membawa fitur-fitur canggih untuk pengembangan web modern.

    2. Sejarah HTML

      3. HTML pertama kali diperkenalkan oleh Tim Berners-Lee pada tahun 1991 di CERN (European Organization for Nuclear Research). Versi pertama HTML, yang disebut HTML 1.0, sangat sederhana dan hanya memiliki sedikit elemen. Kemudian, seiring berjalannya waktu, versi-versi HTML baru diperkenalkan dengan penambahan fitur-fitur baru dan lebih kompleks.

    3. Perkembangan HTML

      4. HTML telah mengalami perkembangan yang signifikan sejak awal mula pada tahun 1991. Dari sederhana versi 1.0, HTML berkembang menjadi HTML5 pada tahun 2014. HTML5 membawa perubahan besar dengan penambahan fitur multimedia seperti audio dan video tags, elemen canvas untuk gambar, dan dukungan untuk animasi serta aplikasi web interaktif. Ini memungkinkan pengembangan halaman web yang lebih menarik dan responsif di berbagai perangkat. Perkembangan HTML masih berlanjut berkat kontribusi komunitas pengembang, memastikan bahwa HTML tetap relevan dan mendukung teknologi web terbaru.
